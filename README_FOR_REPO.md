## What it does

Press a hotkey anywhere in Windows. Get smart AI suggestions powered by Gemini. Insert with Tab.

Works in Notepad, Chrome, Microsoft Teams, Outlook, VS Code, Word — every app you use.

## Seven AI modes

| Hotkey | Mode | What it does |
|---|---|---|
| `Ctrl + Space` | Continue Writing | Autocomplete what you're writing |
| `Ctrl + Shift + Space` | Draft Reply | Generate a reply to selected message |
| `Ctrl + Shift + I` | Improve Writing | Polish grammar, tighten flow |
| `Ctrl + Shift + T` | Translate | Translate to 23 languages |
| `Ctrl + Shift + U` | Summarize | TL;DR of long text |
| `Ctrl + Shift + E` | Explain | Explain jargon, code, concepts |
| `Ctrl + Shift + F` | Fix Code | Detect and fix bugs in code |

## Quick setup

1. **Download** the latest `QuickTyper.exe` from [Releases](../../releases/latest)
2. **Get a free Gemini API key** at [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
3. **Run the .exe**, paste your key, press `Ctrl+Space` in any app
