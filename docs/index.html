Features

Seven AI modes, one hotkey each

Each hotkey runs a different AI prompt on your selected text or context. Works system-wide in any Windows app.

✍️
Continue Writing
Ctrl + Space
Autocomplete what you're writing — emails, docs, anything.
💬
Draft Reply
Ctrl + Shift + Space
Generate a reply to the selected message or email.
✨
Improve Writing
Ctrl + Shift + I
Polish grammar, tighten wording, fix flow of selected text.
🌐
Translate
Ctrl + Shift + T
Translate selected text to any of 23 languages.
📋
Summarize
Ctrl + Shift + U
TL;DR of any long selected text — articles, reports, threads.
🤔
Explain
Ctrl + Shift + E
Explain jargon, code, acronyms, or unfamiliar concepts.
🛠️
Fix Code
Ctrl + Shift + F
Detect and fix bugs in selected code — Python, JS, and more.
Setup

Up and running in 60 seconds

Three steps. You'll need a free Google AI Studio account.

1
Download & run
Click the download button at the top. Run QuickTyper.exe — no installation needed.
2
Get a free Gemini API key
Visit aistudio.google.com/apikey, sign in, click Create API key. Copy it (starts with AIza…).
3
Paste & start
On first launch, paste your key. Press Ctrl+Space in any app — the AI popup appears next to your cursor.
⚠️
Windows might warn you when you first run it
SmartScreen may say "Windows protected your PC" because the app isn't code-signed yet. Click More info → Run anyway. This is normal for indie apps.
